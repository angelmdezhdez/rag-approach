# RAG para papeleria con Google Gemini

Este proyecto implementa un sistema de Recuperación Aumentada por Generación (RAG) utilizando Google Gemini para proporcionar información sobre productos de papelería. El sistema permite a los usuarios realizar consultas y obtener respuestas precisas basadas en una base de datos de productos de papelería.

## Requisitos
- Python 3.8 o superior
- Paquetes necesarios:
  - google-generativeai

## Archivos
En este repositorio encontrarás los siguientes archivos:
- `main.py`: Archivo principal que contiene una generación ásica de respuestas con Google Gemini.
- `data/answers_data.txt`: Archivo de texto que contiene algunas preguntas de ejemplo y sus respuestas correspondientes.
- `data/articles_data.txt`: Archivo de texto que contiene información sobre productos de papelería.
